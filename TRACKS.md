# AEGIS Track Map

- Project root:
  - `aegis/`
  - `cli/`
- Included tracks:
  - Zerion
  - MagicBlock
- Canonical source pages:
  - `resources/track_description_1.md`
  - `resources/track_description_2.md`
- Core positioning:
  - AEGIS is the merged submission for a privacy-first autonomous onchain agent.
  - Zerion is the wallet and execution backbone.
  - MagicBlock provides the private execution and private payment layer.
- Hard constraints:
  - The agent must be built on the forked Zerion CLI.
  - Real onchain transactions are mandatory.
  - Any swap path must respect Zerion's routing requirement.
  - MagicBlock integration must be obvious in the user-facing system, not bolted on.
