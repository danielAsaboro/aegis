## What stayed Zerion

AEGIS did not replace Zerion as the execution backend. Public swaps still use the forked Zerion CLI and Zerion API path for quotes, signing, and broadcast. AEGIS adds the agent layer, guardrails, automation, private route, and interfaces around that base.

## What became safer

The policy boundary is enforced in two places:

- write tools build proposals and run `runPolicies()`
- the executor refuses any proposal that lacks an approved `policyResult`

That means approval UI is an additional control, not the only control.

## What is disclosed as incomplete

MagicBlock shield deposits are verified. The private transfer plus withdraw cycle is blocked by a documented SDK issue after `private: true` deposits. The exact error and reproduction notes are in [Track evidence](/track-evidence) and `TRACKS.md`.
